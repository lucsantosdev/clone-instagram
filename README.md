# 📱 Clone Instagram - Estudo de Caso UI/UX

> Projeto educacional para estudar interface, experiencia do usuario e evolucao visual da tela de login do Instagram.

## ✨ Visao Geral

Este projeto compara duas abordagens da mesma tela:

- 🕰️ **Old Version**: referencia de uma versao mais antiga
- 🚀 **New Version**: proposta evoluida com melhorias de hierarquia, legibilidade e consistencia

📌 O objetivo principal e praticar UI/UX com analise comparativa real.

## 🎯 Proposta do Projeto

Criar um ambiente pratico para:

- 🔍 comparar duas solucoes visuais da mesma tarefa (login)
- 🧠 analisar decisoes de design com base em heuristicas e boas praticas
- 📝 documentar aprendizados de UX writing, acessibilidade e fluxo
- 🛠️ transformar observacoes em melhorias concretas de interface

## ✅ O que ja foi desenvolvido

- 📰 Pagina principal com artigo interativo comparativo
- 🪟 Previews lado a lado da versao antiga e nova
- 🧩 Secoes de analise UX (hierarquia, legibilidade, acoes e consistencia)
- 📊 Matriz comparativa de criterios de experiencia
- 📚 Secao de boas praticas e materiais de estudo
- 📖 Pagina complementar sobre as 10 heuristicas de Jakob Nielsen
- 🌗 Alternancia de tema claro/escuro nas paginas principais

## 🗂️ Estrutura Atual

```text

|- index.html                        # artigo comparativo principal
```

## ▶️ Como executar localmente

1. Abra `index.html` no navegador.
2. Opcional: use Live Server no VS Code para iterar mais rapido.

## 🧠 Objetivos de Aprendizado (UI/UX)

- Hierarquia visual e escaneabilidade
- Densidade informacional e clareza da tarefa
- Consistencia de componentes
- Feedback e previsibilidade de interacao
- Heuristicas de usabilidade (Jakob Nielsen)
- Documentacao de decisoes de design

## 🛣️ Roadmap (Metas Futuras)

### 1. Etapa 1 - Refinamento da Tela de Login 🎛️

- Melhorar estados de foco, erro e sucesso dos campos
- Aumentar cobertura de acessibilidade (ARIA, contraste, teclado)
- Incluir validacoes visuais e microinteracoes mais claras

### 2. Etapa 2 - Fluxos de Autenticacao 🔐

- Criar telas de cadastro, recuperacao de senha e verificacao
- Definir sistema visual compartilhado (tokens e componentes)
- Testar consistencia entre desktop e mobile

### 3. Etapa 3 - Clone Funcional de Areas Principais 🧱

- Home/feed estatico com componentes de post
- Perfil de usuario e pagina de exploracao
- Navegacao entre telas com arquitetura mais robusta

### 4. Etapa 4 - Clone Mais Completo do Instagram 🚀

- Interacoes de curtida, comentario e salvamento
- Simulacao de stories e mensageria basica
- Organizacao por design system + documentacao tecnica
- Integracao com API mock para dados dinamicos
